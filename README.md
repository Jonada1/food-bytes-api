# FoodBytes API

## Requirements
`Docker` should be installed (! Not the docker-toolbox)
`Node` should be installed

## Installation

```bash
npm install
```

## Running the app

### Run Docker First by runing
Windows: `run-docker.bat`
Mac/Linux `run-docker.sh`

### Run Node.js with npm
```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
